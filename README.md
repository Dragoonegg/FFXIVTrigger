## 插件需求  
Triggernometry  
你可能需要额外的插件来保证运行正常  
1：Morelogsline（能获取到更多的log）  
2：PostNamazu（使得你可以向游戏内发送字符串）  

## 安装
在Triggernometry中导入XML文件，并且确保他们启用
